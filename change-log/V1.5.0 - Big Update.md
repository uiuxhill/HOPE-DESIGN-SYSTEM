# V1.5.0 - Token Naming Refactor

## ✅ Completed

### Global Tokens → Numeric Keys
Đổi từ prefix naming sang numeric keys để tránh lặp khi dùng với Tailwind {last version: spacing-space-100)

| File | Before | After |
|------|--------|-------|
| `spacing-global.json` | `"space100": {"value": "8px"}` | `"8": {"value": "8px"}` |

### Dimension Global → Gộp 1 file
Gộp 3 file `fontSize-global.json`, `lineHeight-global.json`, `radius-global.json` thành **1 file duy nhất** `dimension-global.json`.
Build script đọc file này và output 3 bộ CSS vars: `--text-*`, `--leading-*`, `--rounded-*`.

| Before (3 files) | After (1 file) |
|-------------------|----------------|
| `fontSize-global.json` | `dimension-global.json` |
| `lineHeight-global.json` | ↑ |
| `radius-global.json` | ↑ |

Scale: `0, 2, 4, 6, 8, 10 ... 40, 999` (step 2px + 0px + 999px)

### CSS Variable Rename
| Before | After |
|--------|-------|
| `--font-size-*` | `--text-*` |
| `--line-height-*` | `--leading-*` |
| `--radius-*` | `--rounded-*` |

### Brand Tokens → Tailwind-style References
