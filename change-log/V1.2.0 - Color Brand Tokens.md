# V1.2.0 Color Brand Tokens

Tổng hợp các thay đổi cho hệ thống Brand Color Tokens (text, icon, background)

---

## Version 1.2.0 (2026-01-22)

### Files Changed

- `.cursor/rules/color-brand.mdc`
- `tokens/colors/color-brand-light.json`
- `tokens/colors/color-brand-dark.json`

---

### 1. Token Structure - Flat Hyphenated Format

**Thay đổi:** Chuyển từ nested structure sang flat structure với hyphens cho interaction states.

**Trước:**

```json
"subtlest": {
  "value": "{primary.100}",
  "type": "color",
  "hovered": {"value": "{primary.200}", "type": "color"},
  "pressed": {"value": "{primary.300}", "type": "color"}
}
```

**Sau:**

```json
"subtlest": {"value": "{primary.100}", "type": "color"},
"subtlest-hovered": {"value": "{primary.200}", "type": "color"},
"subtlest-pressed": {"value": "{primary.300}", "type": "color"}
```

**Áp dụng cho:**

- `color.link.*`
- `color.background.neutral.*`
- `color.background.brand.*`
- `color.background.{semantic}.*` (information, success, warning, danger, discovery)
- `color.background.selected.*`
- `color.background.input.*`
- `color.background.accent.{hue}.*`

### 2. Link Tokens - Removed Hovered State

**Thay đổi:** Link tokens chỉ có `pressed` state, không có `hovered` state.

**Trước:**

```json
"link": {
  "default": {"value": "{primary.700}", "type": "color"},
  "default-hovered": {"value": "{primary.800}", "type": "color"},
  "default-pressed": {"value": "{primary.900}", "type": "color"},
  "visited": {"value": "{purple.800}", "type": "color"},
  "visited-hovered": {"value": "{purple.900}", "type": "color"},
  "visited-pressed": {"value": "{purple.1000}", "type": "color"}
}
```

**Sau:**

```json
"link": {
  "default": {"value": "{primary.700}", "type": "color"},
  "default-pressed": {"value": "{primary.800}", "type": "color"},
  "visited": {"value": "{purple.800}", "type": "color"},
  "visited-pressed": {"value": "{purple.900}", "type": "color"}
}
```

### 3. Text & Icon Tokens - Renamed Neutral Variants

**Thay đổi:** Đổi tên các token neutral text và icon để rõ ràng hơn.


| Trước      | Sau                |
| ---------- | ------------------ |
| `default`  | `neutral`          |
| `subtle`   | `neutral-subtle`   |
| `subtler`  | `neutral-subtler`  |
| `subtlest` | `neutral-subtlest` |


**Ví dụ (Light mode):**

```json
"text": {
  "neutral": {"value": "{neutral.1000}", "type": "color"},
  "neutral-subtle": {"value": "{neutral.900}", "type": "color"},
  "neutral-subtler": {"value": "{neutral.800}", "type": "color"},
  "neutral-subtlest": {"value": "{neutral.700}", "type": "color"},
  ...
}
```

**Áp dụng cho:**

- `color.text.neutral*`
- `color.icon.neutral*`

---

### 4. Accent Background - Fixed Emphasis Naming

**Thay đổi:** Sửa tên token cho scale 700 từ `bolder` thành `bold` theo đúng rule.

**Rule Emphasis → Scale:**


| Emphasis       | Scale   |
| -------------- | ------- |
| subtlest       | 100     |
| subtler        | 200-300 |
| subtle         | 400     |
| default (omit) | 500-600 |
| **bold**       | **700** |
| bolder         | 800-900 |
| boldest        | 1000    |


**Trước:**

```json
"bolder": {"value": "{red.700}", "type": "color"},
"bolder-hovered": {"value": "{red.800}", "type": "color"},
"bolder-pressed": {"value": "{red.900}", "type": "color"}
```

**Sau:**

```json
"bold": {"value": "{red.700}", "type": "color"},
"bold-hovered": {"value": "{red.800}", "type": "color"},
"bold-pressed": {"value": "{red.900}", "type": "color"}
```

**Áp dụng cho tất cả accent hues:**

- `red`, `orange`, `yellow`, `lime`, `green`, `cyan`, `blue`, `purple`, `magenta`, `gray`

### 5. Special Tokens - Warning Inverse (Text & Icon)

**Thay đổi:** 

- Đổi tên từ nested sang hyphenated format cho text
- Thêm `color.icon.warning-inverse` (giống `color.text.warning-inverse`)


| Token                        | Light            | Dark                |
| ---------------------------- | ---------------- | ------------------- |
| `color.text.warning-inverse` | `{neutral.1000}` | `{darkNeutral.100}` |
| `color.icon.warning-inverse` | `{neutral.1000}` | `{darkNeutral.100}` |

### 6. Neutral Background - Nested Structure with Explicit Default

**Thay đổi:** Chuyển neutral background từ flat structure sang nested structure (giống như `brand`). Giữ lại `default` (không omit).

**Trước:**

```json
"background": {
  "neutral": {"value": "{neutralAlpha.200A}", "type": "color"},
  "neutral-hovered": {"value": "{neutralAlpha.300A}", "type": "color"},
  "neutral-pressed": {"value": "{neutralAlpha.400A}", "type": "color"},
  "neutral-subtle": {"value": "transparent", "type": "color"},
  "neutral-subtle-hovered": {"value": "{neutralAlpha.200A}", "type": "color"},
  "neutral-subtle-pressed": {"value": "{neutralAlpha.300A}", "type": "color"},
  "neutral-boldest": {"value": "{neutral.1000}", "type": "color"},
  "neutral-boldest-hovered": {"value": "{neutral.900}", "type": "color"},
  "neutral-boldest-pressed": {"value": "{neutral.800}", "type": "color"},
  ...
}
```

**Sau:**

```json
"background": {
  "neutral": {
    "default": {"value": "{neutralAlpha.200A}", "type": "color"},
    "default-hovered": {"value": "{neutralAlpha.300A}", "type": "color"},
    "default-pressed": {"value": "{neutralAlpha.400A}", "type": "color"},
    "subtle": {"value": "transparent", "type": "color"},
    "subtle-hovered": {"value": "{neutralAlpha.200A}", "type": "color"},
    "subtle-pressed": {"value": "{neutralAlpha.300A}", "type": "color"},
    "boldest": {"value": "{neutral.1000}", "type": "color"},
    "boldest-hovered": {"value": "{neutral.900}", "type": "color"},
    "boldest-pressed": {"value": "{neutral.800}", "type": "color"}
  },
  ...
}
```

## 7. Border Tokens - (Thêm Mới)

### Summary

Thêm mới property `color.border` với các đặc điểm riêng biệt so với text/icon tokens.

### Cấu trúc

**Neutral Border:**

- `neutral` (omit default): `{neutralAlpha.300A}` (light) / `{darkNeutralAlpha.300A}` (dark)
- `neutral-bold`: `{neutral.600}` (light) / `{darkNeutral.600}` (dark)
- Không có `subtle`, `subtler`, `subtlest`

**Semantic Border:**

- Scale 600 (light) ↔ 500 (dark) - áp dụng mirror scale
- Không có `warning-inverse`

**Accent Border:**

- Chỉ có state default (omit default)
- Scale 600 (light) ↔ 500 (dark)
- Ví dụ: `color.border.accent.red` = `{red.600}` (light) / `{red.500}` (dark)

### Light Mode Example

```json

  "neutral": {"value": "{neutralAlpha.300A}", "type": "color"},
  "neutral-bold": {"value": "{neutral.600}", "type": "color"},
  "brand": {"value": "{primary.600}", "type": "color"},
  "information": {"value": "{information.600}", "type": "color"},
  "success": {"value": "{success.600}", "type": "color"},
  "warning": {"value": "{warning.600}", "type": "color"},

  "discovery": {"value": "{discovery.600}", "type": "color"},
  "inverse": {"value": "{neutral.0}", "type": "color"},
  "disabled": {"value": "{neutralAlpha.200A}", "type": "color"},
  "selected": {"value": "{primary.700}", "type": "color"},
  "accent": {
    "red": {"value": "{red.600}", "type": "color"},
    "orange": {"value": "{orange.600}", "type": "color"},
    "yellow": {"value": "{yellow.600}", "type": "color"},
    "lime": {"value": "{lime.600}", "type": "color"},
```

### Key Differences vs Text/Icon


| Feature           | Text/Icon                                                          | Border                         |
| ----------------- | ------------------------------------------------------------------ | ------------------------------ |
| Neutral emphasis  | `neutral`, `neutral-subtle`, `neutral-subtler`, `neutral-subtlest` | `neutral`, `neutral-bold` only |
| Scale             | 1000-700 (light), 100-400 (dark)                                   | 600 (light), 500 (dark)        |
| `selected`        | 700 (light), 400 (dark)                                            | 700 (light), 400 (dark) - same |
| `warning-inverse` | Có                                                                 | Không                          |
| Accent variants   | `default`, `subtle`, `subtler`, `subtlest`                         | `default` only                 |

## Rules Updated

File `.cursor/rules/color-brand.mdc` đã được cập nhật để phản ánh:

1. **Section 1:** Modifier order với hyphens: `[state]-[emphasis]-[interaction]`
2. **Section 5.1:** Token naming: `neutral`, `neutral-subtle`, `neutral-subtler`, `neutral-subtlest`
3. **Section 5.2:** Neutral Text table với tên token mới
4. **Section 5.4:** `warning-inverse` thay vì `warning.inverse`
5. **Section 6.4:** Neutral Background nested structure với explicit `default`
6. **Section 6.8:** Accent Background với `bold` cho scale 700
7. **Section 7:** Icon tokens với tên neutral mới + `warning-inverse` + accent simplified (chỉ default, scale 600/500)
8. **Section 8:** Link tokens chỉ có `pressed`, không có `hovered`
9. **Section 9:** Border tokens (mới) - neutral, semantic, accent với scale 600; `selected` dùng scale 700/400 như text/icon
10. **Section 10:** Elevation tokens (mới) - surface + shadow cho light/dark
11. **Section 12:** JSON example đã cập nhật (bao gồm border + elevation)
12. **Section 13:** Constraint mới: không dùng nested structure cho interaction states
