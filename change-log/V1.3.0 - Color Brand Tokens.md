## 1. Icon Accent - Simplified Structure

### Summary

Đơn giản hóa `color.icon.accent` tokens - chỉ giữ lại 1 variant duy nhất cho mỗi hue.

### Thay đổi

**Before:**
- `accent.{hue}` (default) - scale 1000/100
- `accent.{hue}-subtle` - scale 900/200
- `accent.{hue}-subtler` - scale 800/300
- `accent.{hue}-subtlest` - scale 700/400

**After:**
- `accent.{hue}` only (omit default) - scale 600/500

### Scale Mapping

| Mode | Before (subtlest) | After |
|------|-------------------|-------|
| Light | 700 | 600 |
| Dark | 400 | 500 |

## 3. Elevation Tokens - Added `surface` + `shadow`

### Summary

Thêm mới `color.elevation` gồm 2 nhóm:
- `color.elevation.surface` (4 levels)
- `color.elevation.shadow` (3 levels)

### Cách dùng (Elevation Surface)

- `elevation/surface/default`: dùng trên **bề mặt bình thường**
- `elevation/surface/raised`: dùng cho **surface nổi lên** (so với `default` hoặc `sunken`)
- `elevation/surface/overlay`: dùng cho **surface của pop-up / modal** nổi lên phía trên
- `elevation/surface/sunken`: dùng cho **mặt phẳng chìm xuống**

### Elevation Surface Mapping

**Light mode**
- `sunken` → `{neutral.100}`
- `default` → `{neutral.0}`
- `raised` → `{neutral.0}`
- `overlay` → `{neutral.0}`

**Dark mode**
- `sunken` → `{darkNeutral.0}`
- `default` → `{darkNeutral.100}`
- `raised` → `{darkNeutral.200}`
- `overlay` → `{darkNeutral.200}`

### Elevation Shadow Mapping

**Light mode**
- `default` → `{neutralAlpha.100A}`
- `raised` → `{neutralAlpha.200A}`
- `overlay` → `{neutralAlpha.300A}`

**Dark mode** (uses `neutralAlpha`, not `darkNeutralAlpha`)
- `default` → `{neutralAlpha.300A}`
- `raised` → `{neutralAlpha.400A}`
- `overlay` → `{neutralAlpha.500A}`

### Token Example (Light)

```json
"elevation": {
  "surface": {
    "sunken": {"value": "{neutral.100}", "type": "color"},
    "default": {"value": "{neutral.0}", "type": "color"},
    "raised": {"value": "{neutral.0}", "type": "color"},
    "overlay": {"value": "{neutral.0}", "type": "color"}
  },
  "shadow": {
    "default": {"value": "{neutralAlpha.100A}", "type": "color"},
    "raised": {"value": "{neutralAlpha.200A}", "type": "color"},
    "overlay": {"value": "{neutralAlpha.300A}", "type": "color"}
  }
}
```

**Files updated:**

- `tokens/colors/color-brand-light.json`

## 4. Accent Background - Added `default` token (scale 500)

### Summary

Bổ sung thêm `color.background.accent.{hue}.default` cho tất cả hue:
- `default` = `{hue}.500`
- `default-hovered` = `{hue}.600`
- `default-pressed` = `{hue}.700`
- `gray.default` = `{neutral.500}` (light) / `{darkNeutral.500}` (dark)

**Note (Dark mode symmetry for accent hues):**
- `default` = `{hue}.600`
- `default-hovered` = `{hue}.500`
- `default-pressed` = `{hue}.400`

**Files updated:**
- `.cursor/rules/color-brand.mdc`
- `tokens/colors/color-brand-light.json`
- `tokens/colors/color-brand-dark.json`
